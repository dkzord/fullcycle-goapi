# 📁 How to download the project

```bash
    #Clone repository
    $ git clone https://github.com/dkzord/fullcycle-goapi.git
```

---

# 📁 How to run the project

```bash
    #Upload the docker application
    $ docker-compose up -d

    #Enter the root folder
    $ cd cmd/catalog

    #Run the project
    $ go run main.go
```
